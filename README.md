# 念佛计数小程序

这是一个基于微信小程序的念佛计数工具，帮助用户记录每日念佛修行，积累福德资粮。

## 功能特性

### 📖 经文计数功能
- 支持多种经文计数（阿弥陀佛、观世音菩萨、地藏王菩萨等）
- 快捷计数按钮（+100、+500、+1000）
- 自定义输入计数功能
- 经文名称和快捷计数可自定义编辑

### 📊 统计功能
- **日统计**：查看当日念佛汇总
- **月统计**：查看本月念佛统计
- **年统计**：查看本年念佛统计
- 详细的分经文统计展示

### 💾 数据管理
- 本地存储，数据安全不丢失
- 可复制当日汇总信息
- 支持单条记录编辑和删除
- 可清空当日所有记录

## 使用方法

### 1. 计数操作
- 点击经文对应的快捷计数按钮快速添加计数
- 在自定义输入框中输入数字后点击"+"按钮添加
- 长按可连续快速计数

### 2. 经文管理
- 点击"新增经文"添加新的经文类型
- 点击经文名称旁的"编辑"按钮修改经文信息
- 点击"删除"按钮移除经文（同时删除相关记录）

### 3. 记录管理
- 在当日汇总区域查看当天的计数记录
- 点击记录旁的"编辑"修改计数
- 点击"删除"移除单条记录
- 点击"清空"删除当日所有记录

### 4. 统计查看
- 切换"日/月/年"选项卡查看不同时间维度的统计
- 统计信息包括总计数和分经文详细计数
