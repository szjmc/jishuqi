# 念佛计数小程序

一个基于微信小程序的念佛计数工具，支持中文和藏文双语，帮助修行者记录每日念佛，积累福德资粮。

## ✨ 功能特性

### 📖 经文计数
- 支持多种经文计数（默认包含皈依、发心、度母）
- 快捷计数按钮（+100、+500、+1000）
- 自定义输入计数功能
- 经文名称和快捷计数可自定义编辑

### 📊 统计功能
- **总计统计**：查看历史累计总数，记录天数和起始日期
- **日统计**：查看当日念佛汇总
- **年统计**：查看本年度念佛统计
- 详细的分经文统计展示

### 🌐 多语言支持
- **中文/藏文一键切换**
- 界面所有文字支持双语显示
- 藏文日期格式转换（使用藏文数字）
- 语言设置自动保存并同步

### 💾 数据管理
- 本地存储，数据安全不丢失
- 可复制当日汇总信息
- 支持单条记录编辑和删除
- 可清空当日所有记录

## 📸 界面预览

### 首页
- 经文计数区域（快捷按钮 + 自定义输入）
- 当日汇总区域（实时显示今日记录）
- 统计概览（日/月/年统计）
- 一键语言切换按钮

### 统计页面
- 总计、日统计、年统计三个选项卡

## 📖 使用说明

### 1. 经文计数
- 点击经文对应的快捷按钮快速添加计数
- 在自定义输入框中输入数字后点击"+"按钮添加
- 快捷按钮按列排列，方便操作

### 2. 经文管理
- 点击"新增经文"添加新的经文类型
- 点击经文名称旁的"编辑"按钮修改经文信息
- 点击"删除"按钮移除经文

### 3. 记录管理
- 在当日汇总区域查看当天的计数记录
- 点击记录旁的"编辑"修改计数
- 点击"删除"移除单条记录
- 点击"清空"删除当日所有记录

### 4. 语言切换
- 点击右上角"藏文/中文"按钮切换语言
- 语言设置会自动保存
- 首页和统计页面语言设置同步

### 5. 统计查看
- 点击底部"统计"标签进入统计页面
- 切换"总计/日统计/年统计"查看不同维度统计
- 统计信息包括总计数和分经文详细计数

## 📊 数据格式

### 经文项目
```javascript
{
  id: '1',                              // 经文ID
  name: '皈依',                         // 经文名称（支持藏文）
  quickCounts: [100, 500, 1000]        // 快捷计数选项
}
```

### 记录数据
```javascript
{
  id: '1',                              // 经文ID
  name: '皈依',                         // 经文名称
  count: 100,                           // 计数
  timestamp: 1708856400000             // 时间戳
}
```

### 每日记录
```javascript
{
  "2026-02-26": [
    { id: '1', name: '皈依', count: 100, timestamp: 1708856400000 },
    { id: '2', name: '发心', count: 500, timestamp: 1708857000000 }
  ]
}
```
